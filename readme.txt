
Πρώτα πρώτα πρέπει να εισαχθούν τα απαραίτητα στοιχεία για να πραγματοποιείται η σύνδεση του προγράμματος με τη βάση
δεδομένων στον localhost(Server:127.0.0.1).
Πρέπει λοιπόν αφου δημιουργήσετε μια νέα βάση στο πρόγραμμα διαχείρισης βάσεων (pgadmin) με sever τον localhost:5432, να συμπληρώσετε το αντίστοιχο πεδίο
της φόρμας με το όνομα της. Επιπλέον για τη δημιουργία σύνδεσης απαιτείται το User ID, το όνομα δηλαδή του χρήστη του προγράμματος διαχείρισης βάσης,
και ο κωδικός με τον οποίο συνδεέεστε στις βάσεις του PostgreSQL (localhost:5432) server μέσω του pgadmin.

Αφού κατασκευάσετε τη βάση στο pgadmin φορτώστε της τα δεδομένα κάνοντας restore το αρχείο MovieDB.
